# Vii-Bot
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/QueenNoviii/vii-bot
> cd vii-bot
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```

---------

### FOR WINDOWS/VPS/RDP USER
```bash
> git clone https://github.com/QueenNoviii/vii-bot
> cd vii-bot
> npm install
```
###### Run
```bash
> node index.js
```
